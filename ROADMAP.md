# Roadmap

## v1.0 — Current Release
- [x] Full 2026 NCAA Tournament bracket (68 teams)
- [x] First Four play-in games with both teams displayed
- [x] Toddler-friendly tap-to-pick UI
- [x] Confetti animations on pick
- [x] Back button, reset, mid-game bracket view
- [x] Printable bracket in traditional mirrored format
- [x] Single self-contained HTML file (all images embedded)
- [x] GitHub Pages hosted

## v1.1 — Mascot Art Upgrade
- [ ] Replace all team logos with cartoon-inspired mascot character illustrations (not official school branding)
- [ ] Consistent art style across all 68 teams (Super Smash Bros / fighting game vibe)
- [ ] AI-generated or commissioned mascot illustrations
- [ ] Ensure all images are original or properly licensed (no school trademark issues)

## v1.2 — Live Bracket Validation
- [ ] Integrate public APIs to pull real tournament results as games go final
- [ ] Compare kid's picks against actual outcomes
- [ ] Score the bracket (ESPN-style scoring: 10/20/40/80/160/320 per round)
- [ ] Visual indicators showing correct picks (green) vs busted picks (red)
- [ ] Running leaderboard if multiple kids fill out brackets

## Future Ideas
- [ ] Support for multiple bracket entries (save/load by name)
- [ ] Women's tournament bracket
- [ ] Sound effects toggle (celebration sounds on pick)
- [ ] Shareable bracket image export (generate a PNG/screenshot of completed bracket)
- [ ] Year-over-year bracket archive
- [ ] Dark mode
- [ ] Accessibility improvements (VoiceOver support, keyboard navigation)
