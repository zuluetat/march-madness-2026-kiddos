# March Madness 2026 Brackets for My Kiddos

A fun, kid-friendly bracket picker for the 2026 NCAA Men's Basketball Tournament. Built for toddlers and young kids to pick winners by tapping on team mascot images.

## Play It Now

**[https://zuluetat.github.io/march-madness-2026-kiddos/](https://zuluetat.github.io/march-madness-2026-kiddos/)**

Works on iPad, iPhone, Android, or any browser. No install needed.

## How It Works

1. Two team mascots appear side-by-side — your kid taps the one they like
2. Confetti celebrates the pick, then the next matchup loads
3. Progress through all rounds: First Four → Round of 64 → 32 → Sweet 16 → Elite 8 → Final Four → Championship
4. After the champion is crowned, view and print the completed bracket

## Features

- **68 teams** with real mascot/character logos (not just text)
- **First Four play-in games** with both teams shown throughout the bracket
- **Back button** to undo picks
- **View Bracket** mid-game to check progress
- **Reset** to start over
- **Printable bracket** in traditional mirrored format (left regions flow right, right regions flow left)
- **Single HTML file** — all 68 images embedded as base64, works offline
- **No external dependencies** — no JavaScript frameworks, no CDN links, no tracking
